# Portfolio Theme System

This file documents the theme tokens currently used by the site.

## Source of truth

Theme variables are defined in `src/index.css` under `:root`.

## Colors

### Backgrounds

- `--bg-0`: `#05070d` (main background)
- `--bg-1`: `#070b14`
- `--bg-2`: `#0b1630`

### Glass surfaces

- `--surface`: `rgba(255,255,255,0.06)`
- `--surface-2`: `rgba(255,255,255,0.08)`
- `--border`: `rgba(255,255,255,0.12)`

### Text

- `--text`: `rgba(255,255,255,0.92)`
- `--muted`: `rgba(255,255,255,0.68)`

### Accent

- `--accent`: `#3b82f6`
- `--accent-2`: `#60a5fa`

## Effects

- `--radius`: `18px`
- `--shadow`: `0 18px 60px rgba(0,0,0,0.45)`

## Stack (project notes)

- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB
